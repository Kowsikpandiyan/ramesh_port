Projects

🏥 Prescripto – Hospital Management System

Prescripto is a full-stack healthcare management platform designed to simplify and digitize hospital operations. It provides an efficient system for patients, doctors, and administrators to manage healthcare services in a seamless and organized manner.


The platform enables patients to book appointments online, view doctor availability, and receive consultation without manual intervention. Doctors can manage their schedules, update patient records, and track appointment history. It also includes an integrated online pharmacy system for ordering medicines and managing prescriptions digitally.


The system is built with a responsive and user-friendly interface to ensure smooth access across devices. Secure payment integration is implemented to support online transactions for appointments and pharmacy orders.

React • Node.js • Express.js • MongoDB • Razorpay • REST APIs • JWT Authentication

💪 Gym Management System

The Gym Management System is a desktop-based application designed to efficiently manage gym operations and automate daily administrative tasks. It helps gym owners and staff handle member registration, trainer allocation, subscription plans, attendance tracking, and performance reports in a structured manner.


The system maintains a centralized database to store member details, membership status, payment records, and workout attendance history. It ensures smooth communication between members and trainers while reducing manual work and improving operational efficiency.


This application also supports generating reports for membership activity, subscription validity, and attendance analysis, helping gym administrators make better decisions.

Java • JDBC • MySQL • Swing (Optional GUI) • OOP Concepts
